# Production Ready Checklist

- [ ] Code is formatted and linted
- [ ] Code is unit tested
- [ ] Code is integration tested
- [ ] Code is end-to-end tested
- [ ] Code is reviewed by a peer
- [ ] Code is deployed to a staging environment
- [ ] Code is deployed to a production environment
- [ ] All secrets are stored in a secure way (e.g. Vault, KMS, etc.)
- [ ] All infrastructure is managed as code (e.g. Terraform, etc.)
- [ ] All dependencies are up to date
- [ ] All vulnerabilities are patched
- [ ] All monitoring and alerting is in place
- [ ] All documentation is up to date
- [ ] All team members are aware of the changes
- [ ] All team members are trained on the new features
